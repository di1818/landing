Программа: 8 встреч, 8 внедрений

  1. Роль HR в бизнесе → определяете зону влияния.
  2. Найм как система → собираете карту воронки.
  3. Интервью → создаёте рабочую модель оценки.
  4. Адаптация → формируете план внедрения.
  5. Обучение → строите базовую систему развития.
  6. Метрики → проектируете HR-дашборд.
  7. Работа с руководителями → усиливаете влияние.
  8. Итоговый проект → защищаете внедрение.
